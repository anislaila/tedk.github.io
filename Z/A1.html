SENSOR

Sensor adalah perangkat yang digunakan untuk mengubah besaran fisika/kimia seperti tekanan, gaya, besaran listrik, cahaya, gerakan, kelembapan, suhu, kecepatan dan fenomena-fenomena lingkungan lainnya yang dapat dimengerti manusia baik melalui perangkat tersebut ataupun ditransmisikan secara elektronik

sensor dapat dikategorikan sebagai transduser input, karena dapat merubah seperti tekanan, gaya, besaran listrik, cahaya, gerakan, kelembapan, suhu, kecepatan menjadi sinyal listrik atau resistansi yang kemudian dikonversikan ke tegangan ataupun sinyal listrik

KLASIFIKASI SENSOR

No. Sensor Keterangan
1. Sensor Aktif Membutuhkan sumber daya eksternal untuk dapat beoperasi.
2. Sensor Pasif Tidak membutuhkan sumber daya eksternal untuk dapat beroperasi.
3. Sensor Analog Dapat menghasilkan sinyal output yang berkelanjutan.
4. Sensor Digital menghasilkan sinyal keluaran diskrit yang akan non-kontinu dengan waktu dan dapat direpresentasikan dalam “bit”.

JENIS SENSOR

No. Sensor Keterangan
1. Sensor Cahaya Mengubah besaran cahaya menjadi besaran listrik.
2. Sensor Tekanan Sensor yang digunakan untuk mengukur jumlah tekanan yang diterapkan pada sebuah sensor.
3. Sensor Suhu Panas Menjadi besaran listrik.
4. Sensor Suara Merupakan sensor analog yang digunakan untuk merasakan tingkat suara.
5. Sensor Kelembapan Sensor yang digunakan untuk melakukan pengukuran kelembaban tanah.
6. Sensor Beban Untuk mengukur tekanan suatu zat. Beban yang diberikan akan mengakibatkan reaksi terhadap elemen logam pada load cells sehingga mengakibatkan perubahan bentuk secara elastis.
7. Sensor Ultrasonik Sensor Ultrasonik bekerja berdasarkan sifat-sifat gelombang suara dengan frekuensi lebih besar daripada rentang suara manusia.
8. Sensor Akselerometer Sensor yang mendeteksi perubahan posisi, kecepatan, orientasi, goncangan, getaran, dan kemiringan dengan gerakan indra.
9. Sensor Magnet Sensor yang akan terpengaruh medan magnet dan akan memberikan perubahan kondisi pada keluaran.
10. Sensor Giroskop Sensor yang digunakan untuk merasakan dan menentukan orientasi dengan bantuan gravitasi bumi.
11. Sensor efek Hall/hall Efect sensor Sensor yang dapat mengubah informasi magnetik menjadi sinyal listrik untuk pemrosesan rangkaian elektronik selanjutnya.
12. Sensor Infrared/Inframerah Sistem akan mentrasmisikan sinyal infrared (IR trasmiter) kemudian sinyal inframerah ini dipantulkan oleh permukaan suatu objek dan sinyal diterima oleh penerima infrared (IR recevier).
